# Conditioners

## Final project from National Program "IT Career"

### How to start the project?

#### For Windows:
```
  pipenv shell
```
```
  pipenv install django django-sslserver pillow
```
```
  python manage.py runsslserver
```

#### For Linux:
```
  pipenv --python 3.10
```
```
  pipenv shell
```
```
  pipenv install django django-sslserver pillow
```
```
  python manage.py runsslserver
```
